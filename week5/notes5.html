Errors, Exceptions, and Warnings

  • Errors are ussually caused by one of the following:
    • System Error - problem with the system or extrernal devices
    • Programmer Error - incorrect syntax, faulty logic or a typo
    • User Error - user has entered data incorrectly
  • An Exception is an error that produces a return value that can then be used by the program to deal with the error
  • An exception will also produe a stack trace which is a sequence of functions or method calls that lead to the point where the error occured
