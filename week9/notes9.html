  • alert(), confirm() and prompt() are all functions of the window object
  • The window.location property is an object that contains info about the URL of the current page.
  • The window.history property can access info about any previously visited pages in the current browser session
  • A new window can be opened using the window.open() method
    • const popup = window.open('https://sitepoint.com','SitePoint',
      'width=400,height=400,resizable=yes');
  • Use the document.cookie property to create a cookie
