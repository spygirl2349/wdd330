Transforms

.ad-ad2 h1:hover span {
color: #484848;
transform: rotate(10deg) translateX(40px) scale(1.5);
}

  • Translation Transforms are similar to position: relative;(it allows you to move elements l, r, up & down
  • Scale is another type of transform
  • The rotate() function rotates an element around the point of origin by a specified angle value
  • The skew(x,y) function specifies a skew along the x and y axes.
  • change the origin by using the transform-origin property

Tranisitions

  • 200ms is considered optimum time for a transition
  • The tranisition-property property lets you chose what property will be affected by the transition
  • tranisition-timing-function property lets you control the pace of the the transition in more granular detail
  • The transition-delay property allows you to add a delay before the transition starts
  • The transition property is shorthand for the four transition properties just described.
  • You can also have multiple transitions by separating them with a comma

.ad-ad2 h1 span {
transition: transform 0.2s ease-out 50ms;
}
