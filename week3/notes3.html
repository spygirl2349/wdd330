Object Methods: this

  • Assign objects a method by doing:
    • myObj.sayHi = function() { alert("Hello!"); };
    • myObj.sayHi(); // Hello!
  • Method shorthand:
    • user = { sayHi() { alert("Hello");} };
  • "this" in methods is similar to "self" in python objects
    • Ex: let user={ name: "John", sayHi(){alert(this.name); } };
    • user.sayHi(); // John
  • In JS, this can be used in any function, even if it's not a method of an object
  • A function can be copied between objects
  • Arrow functions are special: they have no this instead it's taken from outside

JavaScript: Novice to Ninja Ch 5: Objects

  • An object is like a database of values
  • JS objects are similar to a hash or associative array in other programming languages
  • JS Objects are like dictionaries that can contain variables, lists, and functions
  • Create an Object:
    • const spiderman = {};
    • const spiderman = new Object();
  • Access Properties in Objects by using dot notation or bracket notation
    • Pro of bracket notation: superman["real" + " " + "name"] // Clark Kent
  • Use Symbol() to add unique property keys to an object
  • Use thein operator to check if any object has a particular property
  • Alternatively, use the hasOwnProperty() method
  • We can loop through all the object's properties/methods by using the for in loop
  • Use Object.keys() to return an array of all the keys in an object
  • Use Object.values() to return an array of all the values in an object
  • Use Object.entries() to return an array of all the keys-value pairs in an object
  • Adding properties example:
    • superman.city = 'Metropolis';
  • Removing properties: delete superman.fly
  • Nesting Objects:
    • jla.wonderWoman.realName // "Diana Prince"
    • jla.aquaman['realName'] // "Arthur Curry"

JSON

  • JSON (JavaScript Object Notation) was invented by Douglas Crockford in 2001
  • JSON is a string representation of the object literal notation, but there are some differences
    1. Property values must be double-quoted
    2. permitted values are double quoted strings, numbers, true, false, null, arrays, and objects
    3. Functions are not permitted values
  • The parse() method takes a string of data in JSON format and returns a JS object
  • The stringify() method does the opposite, taking a JS object & returning a string of JSON data

The Math Object

  • A built-in object that has several properties representing mathematical constants, it also has methods that perform mathematical operations
  • Examples of math constants:
    • Math.PI // 3.141592653589793
    • Math.E // 2.718281828459045
  • The Math.abs() method returns the absolute value of a number
  • The Math.ceil() method will round a number up to the next integer
  • The Math.floor() method will round a number down to the next integer
  • The Math.pow() method: Math.pow(3, 2); // 3 squared = 9
  • It also does trigonometry: Math.sin() Math.cos()
  • The Math.random() method is used to create random numbers
    • To generate a random integer btw 0 & 6:
    • Math.floor(6 * Math.random())

The Date Object

  • Contains information about dates and times
  • The parameters that can be provided are:
    • new Date(year,month,day,hour,minutes,seconds,milliseconds)
  • Timezones can be tricky. Use UTC
  • Getter Methods
    • Examples: getTime() getTImezoneOffset() getYear()
  • Setter Methods
    • Examples: setDate() setMonth() setFullYear()

The RegExp Object

  • A regular expression (RegExp) is a pattern that can be used to search strings for matches to the pattern
  • Creating regular expressions:
    • const pattern = /[a-zA-Z]+ing$/;
    • const pattern = new RegExp('[a-zA-Z]+ing');
  • RegExp Methods
    • Use the test() method to see if a string matches the RegExp pattern, it will return true or false
    • The exec() method works the same way as test, but returns null if it doesn't match the pattern
  • RegExp Properties
    • The global property makes the pattern return all matches
    • The ignoreCase property makes the pattern case-insensitive.
    • The multiline property makes the pattern multiline
  • Look up for more details on how to use RegExp

JavaScript: Novice to Ninja Ch 6: Document Object Model

  • The Document Object Model (DOM) represents an HTML document as a network of connected nodes that form a tree-like structure
  • Legacy DOM Shortcut Methods
    • Document.body returns the body element of a web page
    • Document.images returns a node list of all the images contained in the document
    • Document.links returns a node list of all the <a> and <area> elements that have an href attribute
    • Document.anchors returns a node list of all <a> elements that have a name attribute
    • Document.forms returns a node list of all the forms in the document
  • Create a Text Node:
    • const flashText = document.createTextNode('Flash');
  • The insertBefore() method will place a new element before another element in the markup
  • The replaceChild() method can be used to replace one node with another
  • Update CSS using the style property
    • superman.style.border = "red 2px solid";
    • superman.style.backgroundColor = 'blue';
    • superman.style.display = 'none';
  • Use the getComputedStyle() function to retrieve all the style information of an element that is given as a parameter
  • You can do: superman.style.boder('red 2px solid'); but it's better to add a class and style the class in a separate css file

JavaScript: Novice to Ninja Ch 7: Events

  • Event Listeners
    • document.body.addEventListener("click", doSomething);
  • Inline Event Handlers
    • <p> onclick="console.log('You Clicked Me!')">Click Me</p>
  • The target will show a message in the console telling us the node that was used
  • Use the mousedown and mouseup events to see when the click button is pushed down or up. It occurs
  • See practice for examples on click, double click, hover, and keyboard events
  • See more on other events such as: touchstart, touchend, touchmove, touchenter, touchleave, touchcancel
  • Use the removeEventListener() method to remove an event listener
  • Event Propagation: Bubbling vs Capturing
    • Bubbling is when the child element fires before the parent element
    • Capturing is when the parent element fires before the child element
    • Bubbling is the default for event listeners, to use capturing do:
      • Add the third parameter to the addEventListener() method. It's a boolean value for whether or not to use capturing
      • ulElement.addEventListener('click', [function goes here] , true)
    • Stop the bubbling phase by adding the event.stopPropagation() method into the callback function
  • Event Delegation
    • Attach an event listener to a parent element to capture events that are triggered by its child elements
    • Use the target property to identify the element that was clicked on
