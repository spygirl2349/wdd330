JavaScript: Novice to Ninja Ch 8: Forms

  • The form.submit() method will submit the form automatically (but it won't trigger the for submit event)
  • The form.reset() method will resert all the form controls back to their initail values
  • The form.action property can be used to set the action attribute of a form (so it's sent to a different URL)
  • Form Events
    • The focus event occurs when an element is focused on
    • The blur event occurs when the user moves the focus away from the form element
    • The change evnt occurs when the user moves the focus away from the form element after changing it
  • Sumbitting a Form (the submit event)

JavaScript: Novice to Ninja Ch 12: OOP

class Turtle {

constructor(name) {
this.name = name;
this.weapon = 'hands';
}

sayHi() {
return `Hi dude, my name is ${this.name}`;
}

attack(){
return `Feel the power of my ${this.weapon}!`;
} }

  • Use static to create a static method, it is not available to instances of the class
  • Create a new instance of a class:
    • const leo = new Turtle('Leonardo');
  • When using classes and prototypes to create instances, the following should be considered:
    • Create a class declaration that deals with any initialization, shared properties and methods
    • Use the prototype to augment extra methods and properties (this includes all instances, even those that have already been created)
    • An instance can be augmented using assignment of properties or methods (invididual to that instance) to that object
    • Be careful when overwriting the prototype completely -- the constuctor class needs to be reset
  • Example
    • const mike = new Turtle('Michelangelo');
    • mike.eat() // 'Mmm, this Pizza tastes great!'
    • mike.weapon = 'Nunchakus';
    • mike.attack(); // 'Feel the power of my Nunchakus!'
