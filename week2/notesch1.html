Week 2: Bonus!

JavaScript: Novice to Ninja, 2nd Edition Ch 1

  • JavaScript was created by Netscape
    • It was orginally called Mocha, then briefly called LiveScript before getting called JavaScript
  • Browser Wars - Netscape vs Microsoft -- Microsoft won
    • A race to add new features to make HTML and JS work together on the web
  • Ajax (Asynchronous JavaScript) and XML:
    • "technique of obtaining data from a server in the background and updating only the relevant parts of the web page without the need for a full page reload, enabling users to continue interacting with the rest of the page"
    • by Jesse James Garrett
  • Node.js aka Node
    • Developed by Ryan Dahl in 2009
    • Allows server-side applications to be written in JS
    • Allows web applications to be written exculsively in JS
    • Isomorphic JavaScript = having JS code run on either the client or server side
  • ECMAScript is the specification while JavaScript refers to the language.
    • They started updating ES every year in June
    • ES6 (ECMAScript Verison 6) == ES2015 (released in 2015)
  • Why install Node?
    1. A lot of tutorials require you to install it
    2. Allows you to use the latest features of the language
    3. you can install a huge variety of tools & code libraries using npm (The Node Package Manager)
    4. You can use it to install Babel a transpiler (convert code written in the latest version of JS into code that will run in most browser)
