JavaScript: Novice to Ninja Ch 2: Programming Basics

  • Make multi-line comments:
    • /* This is a milti-line comment */
  • JS is know as C-style syntax cuz of its similarities with C
  • A block is stuff collected together inside curly braces.
    • { this is a block }
  • Use "typeof" to figure out what type of a value. Ex:
    • typeof 'hello' // 'string'
  • "A ninja programmer will try to keep the number of global variables to a minimum"
  • Strings
    • 'It\'s me' won't cause an error
    • If you want to write a blackslash: "a blackslash \\" it will appear as "a blackslash \"
  • Numbers
    • Check if a value is a number by using Number.isFinite() method
    • Examples of Type Coercion:
      • '2' * 8; << 16
      • '2' + 8; << '28'
    • Undefined is JavaScript's way of saying "I can't find a value for this."
    • Null means 'no value'
    • Null vs Undefined
      • Null behaves like a zero.
      • Undefined is not a number.

JavaScript: Novice to Ninja Ch 3: Arrays, Logic, and Loops

Arrays

  • Removing Values from Arrays
    • delete avengers[3]; << true
  • Destructuring Arrays
    • Allows us to assing multiple values at the same time
    • const [x, y] = [1, 2];
  • Pop, Push, Shift, and Unshift
    • Remove the last item in the array: avengers.pop();
    • Remove the first item in the array: avengers.shift();
    • Append a new value at the end of the array: avengers.push('Thor');
    • Append a new value to the beginning of the array: avengers.unshift('Captain America')
  • Merging Arrays
    • Use the concat() method to merge an array with one or more arrays
    • Use the spread operator. ...
  • Use the join() method to turn an array into a string.
    • avengers.join()
      <<'Captain America, Iron Man, Thor'
    • avengers.join(' & ')
      <<'Captain America & Iron Man & Thor'
  • The slice() method creates a subarray (no items are actually removed from the main array)
  • The splice() method removes items from an array and inserts new items in their place
    • avengers.splice(4, 0, 'Quicksliver'); won't remove any items
    • avengers.splice(3, 1, 'Scarlet Witch'); will remove the item at index 3 and 'Scarlet Witch' will be in it's place.
  • Reverse the order of an array using the reverse() method
  • Sort the order of an array using the sort() method. (Alphabetical order is the default for strings)
  • Use indexOf() method to find the index of an item in an array
  • Use the includes() method to find out whether or not an item is in the list. It will return a boolean value.

Sets

  • A set is a data structure that is like python's dictionary
    • Set { 1, 2, 3 }
  • Create a set by: const list = new Set();
  • Add values to sets: list.add(1);
    • You can't add the same value. It will simply be ignored
  • Examples:
    • Set {[1], [1]}
    • Set {2 , '2'}
